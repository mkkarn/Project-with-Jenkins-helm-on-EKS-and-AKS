# Project-with-Jenkins-helm-on-EKS-and-AKS
Deploy Application on EKS and AKS kubernetes cluster with CI/CD pipeline by using helm chart

> Setup CI with Jenkins, git and maven

*Setup jenkins on AWS

*Setup and configure maven and git in jenkins

*Write Declarative jenkins for CI job

*Create dockerfile

*Run CI job

> Setup CD with Jenkins, helm and k8s

*Write declarative jenkinsfile pipeline for CD

*Setup helm in jenkins

*Managed EKS on AWS

*Managed AKS on Azure

*Create helm charts

*Run CD jons

> Autoscaling of Pods with HPA (Horizontal Pod Autoscaler)

*Configure Metric server with HPA

*Configure autoscaling for Memory utilization

*Configure autoscaling for CPU utilization

> Log Aggregation with EFK (Elasticsearch, Fluentd and Kibana)

*Access pods logs with kubectl

*Configure EFK

*Access logs with Kibana Dashboard

Workflow:

Developer > github > jenkins > Docker registry > helm > AWS EKS

> To install jenkins on server we need to follow below steps

*install java

*Confirm Java version
